# Movie Search (REST API)

Проект для поиска фильмов с использованием внешнего REST API.  
Данные загружаются асинхронно через HTTP-запросы, после чего интерфейс обновляется на основе полученного ответа.

## О проекте

Проект позволяет искать фильмы по названию и просматривать подробную информацию о выбранном фильме.  
Данные загружаются из OMDb API в реальном времени. Результаты поиска отображаются в виде карточек, при выборе фильма открывается отдельная страница с деталями.

Проект выполнен как часть портфолио для стажировки на позицию Frontend Developer.

## Цель проекта

Основной целью проекта было получить практический опыт работы с REST API и асинхронной загрузкой данных.

В рамках проекта реализовано:

- отправка HTTP-запросов к внешнему REST API через `fetch()`
- работа с асинхронными операциями на основе Promise (`then / catch`)
- получение и обработка JSON-ответов
- обновление интерфейса после получения данных
- обработка ошибок на уровне HTTP и ответа API

Проект создавался как практическое применение асинхронной работы в JavaScript и понимания взаимодействия клиентского кода с внешним сервисом.

## Функционал

- поиск фильмов по названию (ввод и нажатие Enter)
- отображение результатов поиска
- переход на страницу фильма по `imdbID`
- загрузка подробной информации о фильме
- отображение состояния загрузки
- вывод сообщений об ошибках

## Используемые технологии

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- REST API (OMDb API)
